Welcome to the library Thin2.0 application.

This showcase application has been built with the following technologies:

Thin2.0

Thin2.0 provides developer tooling and components to easen the development and increase productivity.

Angular 2

Built with Angular2, this example showcases lazy loading and AOT compilation. In the other hand, this exposes the basic Angular2 toolkit.

Webpack

Packaged with Webpack, this example showcases the configuration required in order to provide a development environment together with code quality controls via Karma and Jasmine and an AOT compiled production build.
